# 紧急联系卡(github 托管版）
- 填写基本信息和紧急联系人
- 一键生成大字版“紧急联系卡”
- 生成二维码（扫码即可查看卡片）
- 可打印（放钱包/卡套）
- 数据仅保存在浏览器本地或二维码中，无需服务器

在线演示（部署后替换为你仓库的 Pages 地址）
- 例如：https://你的用户名.github.io/emergency-contact-card/

## 功能
- 表单填写：姓名、年龄、血型、过敏、用药、基础病、住址、联系人（姓名/关系/电话）
- 预览卡片：大字号、高对比度、适老化
- 二维码生成：编码为含数据的链接，别人扫码即可在本页查看
- 打印样式：打印时只显示卡片，适合裁切放置
- 本地保存：自动保存到浏览器 localStorage

## 使用方法
1. 克隆或下载本仓库
2. 直接双击 `index.html` 即可在浏览器使用（离线可用）
3. 或者部署到 GitHub Pages：
   - 新建仓库，推送这些文件
   - 在仓库 Settings -> Pages -> 选择 Branch: main, Folder: /root，保存
   - 打开显示的链接即可在线使用
4. 用手机扫码二维码，可打开同一个页面的“查看模式”，只显示卡片，方便现场查看
5. 点击“打印”按钮，打印出纸质卡片

## 隐私与提示
- 本项目不收集数据，信息仅存储在你的浏览器和二维码/打印纸上
- 二维码被他人扫码后可看到卡片信息，请在安全场景下使用
- 建议打印时隐藏不必要的信息（可删减表单内容后再生成）

## 技术栈
- 纯前端：HTML + CSS + 原生 JavaScript
- 二维码库：qrcode.js（CDN 引入）

## 开发结构
- index.html：页面与结构
- styles.css：样式与打印优化
- app.js：逻辑（表单、预览、二维码、保存/载入）

# 紧急联系卡 - 微信小程序版

本目录包含“紧急联系卡”项目的微信小程序版本源码。该工具旨在帮助用户（尤其是老年人或有特殊病史的人群）快速填写并生成一张包含紧急联系人信息和医疗背景的卡片，支持本地保存和二维码展示，以便在紧急情况下提供必要信息。

## 🌟 主要功能

- **信息录入**：支持姓名、年龄、血型、过敏史、常用药物、基础疾病及居住地址的填写。
- **紧急联系人**：可添加多位紧急联系人，并一键拨打电话（需在真机运行）。
- **实时预览**：填写信息的同时，实时生成大字版“紧急联系卡”预览。
- **本地存储**：利用微信小程序的 `Storage` API，将信息加密保存在手机本地，无需联网，保护隐私。
- **离线使用**：完全不依赖后端服务器，所有逻辑均在前端完成。
## 📂 项目结构

```text
miniprogram/
├── pages/
│   ├── index/          # 主页面：表单填写与卡片展示
│   └── about/          # 关于页面：隐私说明与使用指南
├── utils/              # 工具函数：数据加解密、格式化等
├── app.js              # 小程序逻辑
├── app.json            # 小程序公共配置
├── app.wxss            # 小程序公共样式表
└── project.config.json # 项目配置文件
```

## 🛠️ 技术栈

- **框架**：原生微信小程序 (WXML + WXSS + JavaScript)
- **数据存储**：`wx.setStorage` / `wx.getStorage`
- **交互**：原生微信反馈组件 (`wx.showToast`, `wx.showModal`)

## 🔐 隐私声明

- **数据去向**：本程序产生的所有个人信息**仅保存在用户的手机本地缓存中**。
- **网络访问**：程序不请求任何联网权限，不会将您的个人隐私上传到任何服务器。
